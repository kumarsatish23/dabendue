  • INTRODUCTION : "U DECIPHER"

    Gestures, few consider it is a point of situation where you need to act yourself to make a bridge for communicating with other person. But for some it is the only source of communication.

    The name says it all: U Decipher (meaning Universal translator), the fact that no such hardware has been designed yet makes it unique!

    Itâ€™s a Gesture Recognition based product which not only serves and boosts confidence of speech impaired individuals but also helps them enhance their communication with others.Designed to be worn as a pendant which is light weighed, convenient and stylish.

    This is a wearable device which reads the hand gestures and interprets the hand signs for those people with speech and hearing disability. All the sign gestures are programmed into a mobile application and is connected to the device through a Bluetooth whenever in use. Whenever a Gesture is made it generates an individual code that matches with the codes in the mobile application and Interprets into Words.
